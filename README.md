# Maps

Demo project for using classes with TypeScript

## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/denisjovic/maps.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  parcel index.html
```
